# nodebb-plugin-sso-eveseat

This [NodeBB](https://nodebb.org/) plugin allows users login/register via their [EVE SeAT](http://eve-seat.github.io/) account.

## Install

    $ npm install eve-scout/nodebb-plugin-sso-eveseat
